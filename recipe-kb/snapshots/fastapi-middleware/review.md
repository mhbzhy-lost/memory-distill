## 段落样例
- [fastapi-middleware-1] Middleware ¶
- [fastapi-middleware-2] You can add middleware to FastAPI applications.
- [fastapi-middleware-3] A "middleware" is a function that works with every request before it is processed by any specific path operation . And also with every response before returning it.
- [fastapi-middleware-4] It takes each request that comes to your application.
- [fastapi-middleware-5] It can then do something to that request or run any needed code.
- [fastapi-middleware-6] Then it passes the request to be processed by the rest of the application (by some path operation ).
- [fastapi-middleware-7] It then takes the response generated by the application (by some path operation ).
- [fastapi-middleware-8] It can do something to that response or run any needed code.
